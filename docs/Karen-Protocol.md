## Safety

- Target loss: 0.5-1.0 lb per week
- Gaunt Check: If loss exceeds 2 lbs/week while under 150 lbs, increase calories by 150 kcal
- If the app shows "Warning: Losing too fast", add a high-protein snack

## The Stall Rule (Karen)

Same 3-condition check as Grumpy but with Karen's thresholds:

1. **Weight**: Less than 0.5% body weight loss per week for 3 weeks
2. **Compliance**: 100% of MOTS-c shots logged (no Wolverine check)
3. **Protein**: 100g daily floor consistently met

If all three: "Metabolic Stall Detected. Consider Tirz increase to 1.8mg."
